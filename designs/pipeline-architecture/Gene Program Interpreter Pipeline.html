Selected stage

Biological inputs

The pipeline begins with weighted gene programs, but meaning comes from the experimental frame: organism, tissue, cell type, conditions, assay, and optional Perturb-seq regulator effects.

What enters
  • Gene loading matrix
  • Context profile
  • Regulator effects
What leaves
  • Validated program IDs
  • Resolved biological framing
  • Inputs ready for enrichment

What this architecture teaches: injecting experimental context at every handoff is what makes the interpretation biologically specific; isolating research per program makes the system parallel; preserving a deterministic verification and render path makes the final report auditable and resilient.
